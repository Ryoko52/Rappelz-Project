How to Craft a Medal

Create an Ordinary Medal

Combine one of the following to get a chance at crafting an Ordinary Medal:

  • x1000 Naha's Basic Blueprints
  • x500 Naha's Blueprints
  • x100 Naha's Advanced Blueprints
If it fails, you will receive 1 Medal Shard. Combining 10 Medal Shards will give you an Ordinary Medal.
Alternatively, adding x1 Naha's Advanced Medal Shard to the combination will guarantee success.

Identify the Medal

To identify the medal, combine it with one of the following sets:

  • x1 Philosopher's Scroll + x1 Secret Book
  • or
  • x1 Power of Change: Medal + x1 Secret Book

Enhance the Medal Type

Once identified, combine the medal with a gem to get an enhanced version:

  • Health Gem → Enhanced Health Medal
  • Strength Gem → Enhanced Strength Medal
  • Intelligence Gem → Enhanced Intelligence Medal
It is possible to re-identify Enhanced Medals using a Blessed Power of Change: Medal.

Enchant the Medal

All 4 medal types can be enchanted up to +32 Luciad Power.

  • +0 to +20 requires a Medal Cube or the upgraded Ancient Medal Cube. If the enchantment fails, it can be repaired with an E-Repair Powder: Medal.
  • To increase beyond +20, use Luciad's Power: Medal.

Medal Basic Stats

Each medal type provides the following base stats regardless of enchantment level:

Ordinary Medal
  • +5% Basic Stats (Except Luck)
Enhanced Health Medal
  • +15% Max HP / Max MP
  • +15% P. Def / M. Def
  • +15% HP Recov.
Enhanced Strength Medal
  • +20% P. Atk.
  • +20 Crit. Pow.
Enhanced Intelligence Medal
  • +20% M. Atk.
  • +20 Crit. Pow.
